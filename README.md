# ✨ Overview

ORS ReDesign Version is a modernized Academic Management System that streamlines the management of students, faculty, courses, marksheets, colleges, subjects, and timetables.

The project focuses on:

✅ Modern User Interface

✅ Glassmorphism Design

✅ Responsive Layout

✅ Better User Experience

✅ Clean Navigation System

✅ Professional Dashboard

✅ Role-Based Access Control

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/rishabhshri162/ORS-ReDesign-Version.git
```

### Move to Project

```bash
cd ORS-ReDesign-Version
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
ng serve
```

### Open Browser

```bash
http://localhost:4200
```
